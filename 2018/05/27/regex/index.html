js正则表达式

发表于 2018-05-27 | 分类于 js | 次

正则表达式需要记一下以下规则：
/**

  • 元字符
  • \t 水平制表符 tab
  • \v 垂直制表符
  • \n 换行符
  • \r 回车符
  • \O 空字符
  • \f 换页符
  • \cX 与X对应的控制符（Ctrl + X）
    */
    /**
  • [] 代表一类 [abc]代表匹配abc中的任意一个
  • [^] 代表反向类 [^abc] 匹配除了abc之外的任意值
  • 范围类
  • [a-z] 代表a-z的任意小写字母
  • [A-D] 代表a-z的任意大写字母
  • [a-zA-Z]
  • () 代表一组
    */
    /**
  • 预定义类
  • . 任意字符 [^\n\r]
  • \d 数字字符 (即[0-9])
  • \D 非数字字符 ([^0-9])
  • \s 空白符
  • \S 非空白符
  • \w 单词字符 ([a-zA-Z0-9])
  • \W 非单子字符
    */
    /**
  • 边界字符
  • ^ 以xxx开始
  • $ 以xxx结束
  • \b 单词边界
  • \B 非单词边界
    */
    /**
  • 量词
  • ？ 出现0次或1次
    • 出现一次或多次
    • 出现零次或多次
  • {n} 出现n次
  • {n,m} 出现n到m次
  • {n,} 至少出现n次
    */
    /**
  • 分组 $0永远代表整个表达式
  • /(ab)(cd)(ef)/ 可使用捕获的$1 $2 $3来代表分组
  • ?: 代表被忽略的分组
  • /(?:ab)(cd)(ef)/ 这时捕获的分组只有2组
  • 在表达式内可使用\1 \n来表示匹配的分组
  • /(a)/1/ 代表/aa/
    */
    // 在工作中用到的正则表达式（会不断增加）
